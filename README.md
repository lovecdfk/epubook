# EduRead - Educational EPUB Reader Website

A fully functional educational technology website with an inbuilt EPUB reader that can be deployed on GitHub Pages.

## Features

1. **EPUB File Upload**: Drag-and-drop or browse to upload EPUB files
2. **Inbuilt EPUB Reader**: Custom reader with multiple viewing modes, font controls, and reading features
3. **Educational Library**: Sample educational books categorized by subject
4. **Reading Features**: 
   - Multiple themes (Light, Sepia, Dark)
   - Adjustable font size and family
   - Bookmarks and highlights
   - Table of contents navigation
   - Fullscreen mode
